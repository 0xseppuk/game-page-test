## Структура

```
src/
  app/                    — точки входа Next.js (роутинг, layout, глобальные стили)
  views/
    game-page/            — страница Game Page (композиция виджетов)
  widgets/
    site-header/          — шапка сайта
    site-footer/           — футер сайта
  shared/
    ui/                    — переиспользуемые UI-компоненты (Dropdown, Button, Input и т.д.)
    lib/                   — хуки и утилиты
    styles/                — глобальные SCSS-переменные и миксины
    config/                — конфигурация и константы
    assets/                — статические ресурсы (шрифты)
```
